# VPC Tear Down

## 1. Terminate instances inside the VPC

#### Click "EC2"

- Select instances
- Terminate instances

## 2. Delete the Security Groups you made in the VPC

#### Click "VPC"

#### Click Security Groups

- Select security groups
- Click "Actions"
  - Delete security groups

## 3. Delete the VPC

#### Click "Your VPCs"

- Select VPC
- Click "Actions"
  - Delete VPC
  - NB: Will delete VPC as well as route tables, internet gateway and subnets
  - Type: delete
  - Delete
